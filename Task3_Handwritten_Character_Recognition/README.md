# ✍️ Task 3 - Handwritten Character Recognition

## 🤖 Project Overview
This project is a Streamlit-based AI system that recognizes handwritten alphabets (A-Z) using a CNN model trained on EMNIST dataset.

---

## 🚀 Features
- Upload handwritten image
- Predict alphabet (A-Z)
- CNN deep learning model
- Image preprocessing with OpenCV
- Streamlit interactive UI

---

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Streamlit
