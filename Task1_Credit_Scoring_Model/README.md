# Task 1: Credit Scoring Model

## Objective
Predict an individual's creditworthiness based on financial data using Machine Learning.

## Dataset Features
- Age
- Annual Income
- Number of Bank Accounts
- Number of Credit Cards
- Loan Details
- Payment History

## Approach
1. Data Cleaning and Preprocessing
2. Feature Engineering
3. Label Encoding
4. Model Training using Random Forest
5. Model Evaluation (Accuracy, Precision, Recall, F1 Score)

## Model Used
- Random Forest Classifier

## Results (Example)
- Accuracy: ~79%
- Precision: ~79%
- Recall: ~79%
- F1 Score: ~79%

- Confusion Matrix 
- Feature Importance
